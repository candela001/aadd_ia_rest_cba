Proyecto Java (Maven) que implementa un cliente REST para consumir una API de Inteligencia Artificial mediante peticiones HTTP.
Permite enviar un prompt en formato JSON a un modelo de IA y procesar la respuesta recibida desde el servicio remoto.
Su objetivo es practicar el uso de `HttpClient`, el consumo de APIs REST y la integración de servicios externos en aplicaciones Java.

Examen 12/02/2026 realizado por Candela Búrdalo Arcones.
